# Handoff: Дерево каталога проекта (панель «ПРОЕКТ»)

## Overview
Переработка файлового дерева в панели **ПРОЕКТ** редактора DSL Studio. Цель — заменить «убогий» вид (emoji-иконки, текстовые треугольники) на чистую IDE-навигацию: двухтоновые иконки папок, документы с цветовой кодировкой по типу файла, шевроны раскрытия, направляющие линии отступов, ховер и акцент выделения.

## About the Design Files
Файлы в этом бандле — **дизайн-референс, сделанный в HTML**. Это прототип, показывающий нужный внешний вид и поведение, **а не код для прямого копирования**. Целевое приложение — **Qt** (DSLRay). Задачу нужно реализовать средствами Qt по существующим паттернам проекта:
- **QML**: `TreeView` (Qt Quick) с кастомным `delegate`, либо `ListView` по уплощённой модели; иконки — как `Image`/`Shape`(`PathSvg`) или предрендеренные SVG-ресурсы.
- **Widgets**: `QTreeView` + кастомный `QStyledItemDelegate` (рисование иконок, отступов, направляющих, выделения в `paint()`), стилизация через QSS.

Все размеры ниже — в логических px (DIP); умножайте на `devicePixelRatio` при растеризации иконок.

## Fidelity
**High-fidelity (hifi).** Финальные цвета, типографика, отступы и состояния. Воспроизводить попиксельно средствами Qt.

## Screens / Views
### Панель «ПРОЕКТ»
- **Purpose**: навигация по файлам/папкам проекта, выбор и открытие файла во вкладку, drag-and-drop перемещение, создание папки/файла.
- **Layout**: вертикальный список строк фиксированной высоты внутри прокручиваемой области. Шапка панели (заголовок + 2 кнопки действий) сверху, список — ниже со скроллом.

#### Шапка панели
- Текст `ПРОЕКТ`: 11.5px, weight 700, letter-spacing 0.7px, color `#9aa0ad`, uppercase.
- Справа — 2 квадратные кнопки 26×26px, border `1px #e6e8ed`, radius 6px, фон `#fff`. Внутри SVG-иконка 15×15 «папка с плюсом» / «документ с плюсом» (см. ниже).

#### Строка дерева (узел)
- **Высота строки**: 25px, выравнивание по центру по вертикали.
- **Отступ слева (indent)**: `8 + depth*15` px (depth — глубина вложенности, корень = 0).
- **Внутренние элементы строки слева направо, gap 6.5px**:
  1. **Дисклоужер** (только для папок): шеврон-иконка 13×13 (см. ниже). Для файлов — пустой спейсер 13px (чтобы иконки выравнивались).
  2. **Иконка** 16×16: папка (двухтоновая) или документ (цвет по типу).
  3. **Имя**: 13px, letter-spacing -0.05px. Папки и файлы: weight 500; выделенный узел: weight 600. Обрезка по `text-overflow: ellipsis`, `white-space: nowrap`.
- **Radius строки**: 6px.

#### Направляющие отступов (indent guides)
Для каждого предка узла рисуется вертикальная линия:
- толщина 1px, цвет `#e8eaef`, на всю высоту строки.
- X-координата линии для уровня `k` (0..depth-1): `14.5 + k*15` px от левого края строки.
- У **выделенной** строки направляющие не рисуются (`transparent`).

## Interactions & Behavior
- **Hover** (невыделенная, не drop-таргет): фон строки `#f3f4f8`, transition `background .1s`. Уход курсора — фон `transparent`.
- **Click**:
  - на папке → переключает раскрытие (open/closed) и делает её выделенной.
  - на файле → открывает вкладку с этим файлом (дедуп по имени) и делает выделенным.
- **Selected**: фон `#eef1fe` (accSoft), цвет текста = accent `#4b5bd6`, иконки перекрашиваются в accent, слева inset-полоса `inset 2.5px 0 0 #4b5bd6`.
- **Disclosure-анимация**: шеврон поворачивается `rotate(90deg)` при раскрытии, transition `transform .14s ease`.
- **Drag-and-drop**: строки `draggable`. При наведении на drop-таргет — фон `rgba(75,91,214,0.16)` (accRing) + рамка `inset 0 0 0 1px #4b5bd6`. Drop на папку → вложить внутрь (папка раскрывается); drop на файл → вставить рядом, в тот же родительский список. Запрет drop узла внутрь самого себя/потомка.
- **Создание**: кнопка-папка добавляет «Новая папка» (open), кнопка-файл добавляет `untitledNN.dsl`; новый элемент кладётся внутрь выделенной папки, либо рядом с выделенным файлом, либо в корень.

## State Management
- **Модель дерева**: рекурсивная — узел `{ type: 'folder'|'file', name: string, open?: bool, children?: node[] }`. Каждый узел адресуется путём-массивом индексов от корня (`path`).
- **selPath**: путь выделенного узла (или null).
- **dropProj**: путь текущего drop-таргета во время DnD (или null).
- **open** на папках: состояние раскрытия (персистентно в рамках сессии).
- При раскрытии/сворачивании папки перестраивается уплощённый список видимых строк.

## Assets
Иконки не растровые — все построены как inline-SVG пути (см. выше). Растровых ассетов нет. Для Qt: либо `PathSvg`, либо экспорт каждого пути в `.svg`-ресурс. Шрифт прототипа — Hanken Grotesk; в Qt используйте системный/проектный UI-шрифт, метрики близки.

## Files
- `DSL Studio.dc.html` — полный HTML-прототип редактора. Логика дерева — методы `renderProjectRows`, `svgFolder`, `svgFile`, `svgChevron`, `fileTint` в классе `Component`; данные дерева — поле `project` в стейте. Кнопки шапки панели — в шаблоне рядом с заголовком `ПРОЕКТ`.
